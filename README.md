# RecycleView - GithubUser With API

## Features
- [✓] Display Data using RecycleView
- [✓] Details page for each data that will contains
```bash
		[✚] Avatar
		[✚] Username
		[✚] GitHub Profile Link
```
- [✓] Search By User
- [✓] Loading
